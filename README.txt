README

1. Make sure you have apache server setup 
	(You can check this by opening your browser and type in 'localhost'. If you see the apache test page you are golden)

2. Install node.js

3. Open the server.js file and configure it with your password, email, username etc...

4. Open terminal and run the server.js script (command : nodejs server.js)

5. Open another terminal and run the 60_sec_interval.sh bash script (command : sh 60_sec_interval.sh)
	NOTE: You may have to give read, write and execute access to the urlpenetrator.sh bash script

6. Do not close those 2 terminals

7. When grades have arrived you will receive an email
